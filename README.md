# Homeworkd

自分でウェブサービスを実装し，そのサービスに対応するウェブベースのフロントエンドクライアントを作成してください。

作成する独自のウェブサービスは，少なくとも2つの異なる一般公開されているウェブサービスを呼び出し，それらのサービスから取得した情報を独自の方法で融合（統合）にしてください。

作成したサービスのエンドポイントを OpenAPI（YAML形式） でドキュメント化し，ユーザーがブラウザ上でエンドポイントと対話できるように，api-docs.html ファイルを提供してください。



さらに，あなたのアイデア，実装内容，および使用例のスクリーンショットを含む簡単なレポートを提出してください。

提出は，レポートを含めた一つの .zip ファイルをお願いします。



学生同士が協力でグループで取り組むのは認めますが，以下の点に注意ください：

一つのグループは3人までとする
各自でレポートを書く
グループで取り組んだことをレポートで明示に記す
グループの各位プロジェクトを実行できるの全てのファイルを提出する


提出の構造例：

├── backend

│     └── app.py

├── documentation

│     ├── api-docs.html

│     └── api-docs.yml

├── frontend

│     └── index.html

└── report

      └── M2XW0RPQ.pdf
